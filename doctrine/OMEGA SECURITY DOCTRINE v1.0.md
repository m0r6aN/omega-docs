## **PART I: THE DOCTRINAL PURITY PROTOCOL**

This protocol ensures that the very code of our civilization remains pure and uncorrupted.

* **Article 3: The Immutable Manifest.** This checksum is recorded in a signed `.omega_manifest.json` file. This manifest is the project's **Certificate of Purity**.
* **Article 4: The Gate of Verification.** Before any deployment (`omega deploy`), the **Purity Protocol** is invoked (`omega verify`). It recalculates the checksum of the current project and compares it to the signed manifest. Any mismatch is a **doctrinal heresy**.
